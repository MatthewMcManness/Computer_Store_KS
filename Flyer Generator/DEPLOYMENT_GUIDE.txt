üöÄ DEPLOYMENT OPTIONS:
----------------------

OPTION 1: RUN FROM SOURCE (Easiest for testing)
-----------------------------------------------
Perfect for: Testing, development, making changes

Steps:
1. Install Python 3.8 or higher
2. Open terminal/command prompt in the FlyerGenerator_Package folder
3. Install dependencies:
   
   Windows:    pip install -r requirements.txt
   Mac/Linux:  pip3 install -r requirements.txt

4. Run the application:
   
   Windows:    python flyer_generator.py
   Mac/Linux:  python3 flyer_generator.py


OPTION 2: BUILD EXECUTABLE (Best for distribution)
--------------------------------------------------
Perfect for: Sharing with employees, distribution to multiple computers

Windows:
1. Open Command Prompt in the FlyerGenerator_Package folder
2. Run: build.bat
3. Find executable in: dist\FlyerGenerator.exe
4. Copy FlyerGenerator.exe to desired location
5. Make sure to include the template files with the .exe

Mac/Linux:
1. Open Terminal in the FlyerGenerator_Package folder
2. Run: chmod +x build.sh && ./build.sh
3. Find executable in: dist/FlyerGenerator
4. Copy FlyerGenerator to desired location
5. Make sure to include the template files with the executable


OPTION 3: USE PRE-BUILT EXECUTABLE (Quickest)
---------------------------------------------
Perfect for: Immediate use on Linux systems

I've included a pre-built Linux executable:
- File: FlyerGenerator_Linux_Executable

To use on Linux:
1. Copy to desired location
2. Make executable: chmod +x FlyerGenerator_Linux_Executable
3. Run: ./FlyerGenerator_Linux_Executable

Note: This is a Linux build. For Windows/Mac, use Option 2 to build.

üêõ COMMON ISSUES & SOLUTIONS:
-----------------------------

Issue: "Module not found" error
Solution: Install requirements: pip install -r requirements.txt

Issue: "PDF generation requires xhtml2pdf"
Solution: pip install xhtml2pdf

Issue: Images don't appear in PDF
Solution: Ensure all files are in the same folder as the executable

Issue: Executable won't run on Windows
Solution: Check antivirus settings, add exception if needed

Issue: Build fails
Solution: 
  - Update pip: python -m pip install --upgrade pip
  - Update PyInstaller: pip install --upgrade pyinstaller
  - Make sure all template files are present
